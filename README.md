# 2048 / 2248 / Knoppenspel

Browser games built with Phaser 3: **2048** (swipe merge), **2248** (path link merge), and **Knoppenspel** (binary matching).

## Play online

**[Play the published game →](https://jelleAtEijkelkamp.github.io/stunning-tribble/)**

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:5173 and pick a game from the menu.

| Command | Purpose |
|---------|---------|
| `npm test` | Run Vitest unit tests |
| `npm run build` | Production build to `dist/` |

See [AGENTS.md](./AGENTS.md) for the full workflow, file map, and TDD guidelines.
